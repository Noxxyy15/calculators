# Multi-Language Calculator

A simple calculator implemented in 5 different programming languages, just for fun.

## Languages

- **C++** - `calc.cpp`
- **Go** - `calc.go`
- **Lua** - `calc.lua`
- **Python** - `calc.py`
- **Rust** - `calc.rs`

## Features

Basic arithmetic operations:
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)

## Usage

### C++
```bash
g++ calc.cpp -o calc
./calc
```

### Go
```bash
go run calc.go
```

### Lua
```bash
lua calc.lua
```

### Python
```bash
python calc.py
```

### Rust
```bash
rustc calc.rs -o calc
./calc
```

## Requirements

Just have the respective language/compiler installed on your system.
